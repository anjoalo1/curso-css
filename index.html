CURSO CSS

Que es css?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación y el aspecto visual de un documento HTML o XML. Es una tecnología fundamental en el desarrollo web y permite controlar la apariencia de los elementos en una página web, como el color de fondo, el tamaño de la fuente, la disposición de los elementos, los márgenes, el espaciado y muchos otros aspectos visuales.

CSS trabaja en conjunto con HTML para separar la estructura y el contenido de un documento web de su estilo. Mientras que HTML se encarga de definir los elementos y la estructura de una página web, CSS se utiliza para aplicar estilos y diseñar cómo se mostrarán esos elementos en la pantalla.

Como lo agrego a mi proyecto

En el proyecto donde tengas tu codigo html, por lo general creas una carpeta nueva de nombre styles, dentro de esta carpeta crear un archivo de extension css.

Para agregar CSS a un proyecto web, puedes seguir estos pasos generales:

Crear un archivo CSS: Primero, crea un archivo CSS separado con extensión ".css". Puedes nombrarlo como desees, por ejemplo, "style.css". Este archivo contendrá todas las reglas y estilos que deseas aplicar a tu proyecto.

Vincular el archivo CSS: Abre tu archivo HTML en un editor de texto y agrega una etiqueta en la sección . Dentro de la etiqueta , establece el atributo "href" para apuntar al archivo CSS que creaste en el paso anterior. Por ejemplo:

Asegúrate de que el atributo "href" apunte correctamente al archivo CSS y esté en la misma ubicación o ruta relativa adecuada. 

              
                <html>
                  <head>
                  <!-- href ="aqui entre las se coloca la ruta del archivo" -->
                    <link rel="stylesheet" href="styles/style.css"> 
                  </head>
                  <body>
    
                  </body>
                </html>
