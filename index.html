CURSO CSS

¿Que es css?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación y el aspecto visual de un documento HTML o XML. Es una tecnología fundamental en el desarrollo web y permite controlar la apariencia de los elementos en una página web, como el color de fondo, el tamaño de la fuente, la disposición de los elementos, los márgenes, el espaciado y muchos otros aspectos visuales.

CSS trabaja en conjunto con HTML para separar la estructura y el contenido de un documento web de su estilo. Mientras que HTML se encarga de definir los elementos y la estructura de una página web, CSS se utiliza para aplicar estilos y diseñar cómo se mostrarán esos elementos en la pantalla.

¿Como lo agrego a mi proyecto?

En el proyecto donde tengas tu codigo html, por lo general creas una carpeta nueva de nombre styles, dentro de esta carpeta crear un archivo de extension css.

Para agregar CSS a un proyecto web, puedes seguir estos pasos generales:

Crear un archivo CSS: Primero, crea un archivo CSS separado con extensión ".css". Puedes nombrarlo como desees, por ejemplo, "style.css". Este archivo contendrá todas las reglas y estilos que deseas aplicar a tu proyecto.

Vincular el archivo CSS: Abre tu archivo HTML en un editor de texto y agrega una etiqueta en la sección . Dentro de la etiqueta , establece el atributo "href" para apuntar al archivo CSS que creaste en el paso anterior. Por ejemplo:

Asegúrate de que el atributo "href" apunte correctamente al archivo CSS y esté en la misma ubicación o ruta relativa adecuada. 

              
                <html>
                  <head>
                  <!-- href ="aqui entre las se coloca la ruta del archivo" -->
                    <link rel="stylesheet" href="styles/style.css"> 
                  </head>
                  <body>
    
                  </body>
                </html>

Esctructura CSS

Las reglas CSS se utilizan para aplicar estilos a los elementos HTML en una página web. Cada regla CSS consta de dos partes principales: un selector y un bloque de declaración. Aquí tienes la estructura básica de una regla CSS:

Selector: El selector apunta a los elementos HTML a los que se aplicarán los estilos. Puede ser el nombre de un elemento, una clase, un identificador, un atributo o incluso una combinación de ellos. Aquí tienes algunos ejemplos:

  • Nombre de elemento: p selecciona todos los elementos <p>.
  • Clase: .clase selecciona todos los elementos con la clase CSS "clase".
  • Identificador: #id selecciona el elemento con el identificador CSS "id".

Bloque de declaración: El bloque de declaración está contenido entre llaves { } y contiene una o más declaraciones separadas por punto y coma ;. Cada declaración consta de una propiedad y su valor, separados por dos puntos :. Aquí tienes un ejemplo: 

              
                selector {
                  propiedad1: valor1;
                  propiedad2: valor2;
                }

Las propiedades representan los aspectos que deseas cambiar, como el color del texto, el tamaño de la fuente, el margen, el fondo, etc. Los valores son los ajustes específicos que deseas aplicar a esas propiedades, como "rojo" para el color del texto, "14px" para el tamaño de la fuente, etc.

Por ejemplo: 

              
                p {
                  color: red;
                  font-size: 14px;
                  margin-top: 10px;
                }

En este caso, se aplicarían los estilos a todos los elementos <p>, estableciendo el color del texto en rojo, el tamaño de la fuente en 14 píxeles y el margen superior en 10 píxeles.

Comentarios En CSS

Un comentario de código CSS es una línea o bloque de texto que se utiliza para explicar o documentar partes del código CSS. Estos comentarios no afectan la forma en que se renderiza el sitio web, ya que son ignorados por el navegador.

Los comentarios de CSS son útiles para hacer notas sobre el propósito o la función de ciertas reglas CSS, así como para proporcionar información adicional a otros desarrolladores que pueden trabajar en el mismo código más adelante.

< class="parrafo"> Hay un tipo de comentarios en CSS:

Se inician con /* y se cierran con */. Todo lo que se encuentra entre estas marcas se considera un comentario y no se ejecutará. Por ejemplo: 

                

                    /*
                    Este es un comentario y no
                    será tenido en cuenta por el navegador
                    */


                    /*Aquí estoy aplicando un estilo a la clase nombre,
                    el color es azul*/
                    .nombre{
                      color: blue;
                    }

Recuerda que esta es solo una estructura básica, y CSS ofrece muchas más opciones y propiedades para personalizar los estilos de tus elementos HTML.
